# task01
№15

Вводится целое трехзначное число X (в виде int). Если данное число распечатать, то будет напечатано ABC (С !=  '0'). Получить из числа X число Y (в виде int) перестановкой цифр числа так, чтобы при распечатки получилось число CBA. Например, из 123 должно получиться 321, из 778 → 877 и т.д. Строки для конвертации числа не использовать (можно использовать только переменные типа int).
